Async Weather Tracker
This is a simple web application that provides real-time weather information for cities around the world. It uses the OpenWeatherMap API to fetch current weather data, providing users with essential details such as temperature, humidity, and wind speed.

Features
Weather Search: Users can search for any city to get its current weather information.
Detailed Data: The application displays the temperature in Celsius, humidity percentage, and wind speed in meters per second.
Search History: All successfully searched cities are saved in the local storage, allowing users to quickly access them again by clicking on history buttons.
Error Handling: Displays clear error messages if a city is not found or if there is an issue with the API request.
Technology Stack
HTML: For the application structure.
CSS: For styling and layout.
JavaScript: Handles API requests, search history, and dynamic content updates.
OpenWeatherMap API: Source for real-time weather data.
How to Use
Clone or download the repository to your local machine.
Open the weather.html file in any modern web browser.
Enter the name of a city in the search input field and click the Search button.
View the weather details in the Weather Info section on the right.
Access your previous searches through the Search History section on the left.
Project Structure
weather.html: The main entry point of the application.
style.css: Contains all the visual styles for the user interface.
weather.js: Manages the search logic, API calls, and local storage history.
